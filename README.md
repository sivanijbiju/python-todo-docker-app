# Dockerized Python Todo App

## 📌 Project Description
This is a simple Todo web application built using Python Flask and Docker.

## 🚀 Features
- Add tasks
- Delete tasks
- Runs inside Docker container

## 🛠️ Technologies Used
- Python (Flask)
- Docker
- Docker Compose

## ⚙️ How to Run

```bash
# 1. Clone the repository
git clone https://github.com/sivanijbiju/python-todo-docker-app.git

# 2. Navigate into project
cd python-todo-docker-app

# 3. Run the application
docker-compose up

# 4. Open in browser
http://localhost:5000
```
